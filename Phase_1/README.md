# Phase 1: Conceptualization & Initial Design (Months 1 & 2)

This phase covers the foundational research, theoretical kinematics, and preliminary digital design for AmphiGlide's biomimetic locomotion system.

## 📂 Directory Structure

* **`/CAD_Files`**: Contains early 3D models, assembly drafts, and `.step` files for the core drive mechanism and internal chassis.
* **`/Code`**: Initial pinout planning, baseline communication scripts for the Raspberry Pi 4 to ESP32 interface, and preliminary servo testing logic. 
* **`/Images`**: Reference photos of the initial electronic components, early whiteboard sketches, and preliminary computational fluid dynamics (CFD) simulation screenshots.

## 🎯 Phase Milestones Achieved

1. **Mechanical Design:** Drafted the preliminary mechanical linkages required to translate standard servo actuation into the undulating, sinusoidal movement needed for the fin.
2. **Hardware Architecture:** Finalized the selection of core processing units (Raspberry Pi 4 and ESP32) and mapped out the theoretical load for the 4-bus power distribution system.
3. **Simulation Planning:** Established the baseline parameters for testing fluid-structure interaction before moving to physical rapid prototyping.
